AnythingLLM · 온프레미스 챗봇 위젯 데모

사내 웹에 챗봇 붙이기

이 페이지는 AnythingLLM 임베드 위젯을 이용해 온프레미스 LLM 챗봇을 실행하는 최소 예제입니다. 아래 버튼을 누르거나 우측의 코드대로 적용하면 사이트 어디서든 대화가 가능합니다.

실서비스 시에는 로컬호스트 대신 사내 도메인과 HTTPS를 사용하세요.

빠른 확인 체크리스트

  • AnythingLLM 서버가 동작 중이며 /api/embed 엔드포인트가 응답하는가?
  • 위젯 JS 파일 경로(/embed/anythingllm-chat-widget.min.js)가 외부에서 접근 가능한가?
  • data-embed-id가 올바른 워크스페이스 ID인가?

설치 방법

  1. 아래 스니펫을 </body> 바로 위에 붙여넣습니다.
  2. data-base-api-url과 src에 사용 중인 서버 주소(도메인/포트)를 입력합니다.
  3. 필요 시 도메인에서 HTTPS와 리버스 프록시(NGINX)를 설정합니다.
<!-- AnythingLLM Embed (https://anythingllm.com) -->
<script
  data-embed-id="52e2353e-b70c-47be-bd8f-7c7fded26508"
  data-base-api-url="http://localhost:3001/api/embed"
  src="http://localhost:3000/embed/anythingllm-chat-widget.min.js">
</script>
<!-- /AnythingLLM Embed -->

배포 환경에서는 localhost 대신 https://your-internal-domain 형식으로 교체하세요.
